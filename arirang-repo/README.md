## How to publish on GitHub Pages

1. Go to [github.com](https://github.com) and sign in (or create a free account)
2. Click the **+** button (top right) → **New repository**
3. Name it: `arirang-website`
4. Set it to **Public**
5. Click **Create repository**
6. Click **uploading an existing file**
7. Drag and drop **both files** from this folder:
   - `index.html`
   - `README.md`
8. Click **Commit changes**
9. Go to **Settings** → **Pages** (left sidebar)
10. Under **Source**, select `main` branch → click **Save**
11. Wait 1–2 minutes — your site will be live at:

```
https://YOURUSERNAME.github.io/arirang-website
```

## How to add to Google Business Profile

1. Go to [google.com/business](https://google.com/business)
2. Find and claim **Arirang Dumpling & Catering**
3. Go to **Edit Profile → Website**
4. Paste your GitHub Pages URL
5. Save ✅
